## Abstract

We report the discovery of a super-integrable dynamical regime within a carry-coupled extension of the Collatz map. While standard chaotic maps typically exhibit Wigner-Dyson spectral statistics under coupling, we demonstrate that this system maintains Poissonian or Hyper-Crystalline spectral statistics (Level Spacing Variance >> 1) even under non-linear fiber-fiber interaction.

We identify the mechanism as **Arithmetic Lattice Locking** - a phenomenon where the modular carry gate enforces a discrete rigidity that forbids the level repulsion associated with quantum chaos. This results in a degenerate ground state (K=4) capable of storing approximately log₂(p/4) bits of information with neutral stability.

The system effectively acts as a **Decoherence-Free Subspace** naturally emergent from integer arithmetic, realizing an **Arithmetic Random Access Memory (RAM)** distinct from quantum chaotic attractors typically sought in number theoretic physics.

Standard Collatz dynamics `(3n+1)/2` treat the division by 2 as a lossy operation where fractional information vanishes.

### The Hypothesis

We proposed that this "lost fraction" is not discarded but is **carried** into a hidden fiber space, creating a coupled system.

### The Construction

We defined a skew-product transformation:

```
T: Z x Z_p --> Z x Z_p
```

Where:
- **Base**: The Collatz map acting on w (the integer)
- **Fiber**: A modular state n in Z_p (the phase)
- **Coupling**: The fiber update depends on the CARRY generated by the base

## Mathematical Framework

### The Skew-Product Transformation

For coupling constant K and prime modulus p:

```
Fiber Update:    n' = (K * n) mod p
Carry:           c  = floor(K * n / p)
```

### The Identity Condition

**Theorem**: For K=4, the return map becomes the identity operator:

```
R_4(n) = n (mod p)
```

This is not chaos - it is a **ground state**.

### Safe Window

States with zero carry form the "safe window":

```
W_4 = { n : 0 <= n < p/4 }
```

This window contains approximately 25% of the state space.
